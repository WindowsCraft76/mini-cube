# Privacy Statement – ​​Mini Cube

**Last updated: February 22, 2026**

Your privacy is important. Mini Cube does not collect **any personal data**.

## 1. Data Collected and Storage
- The following information is **stored locally** on your computer solely to enable the game to launch: Minecraft Username, UUID, Authentication Token, and others.
- This information is **only accessible by Mini Cube** to launch the game.
- Mini Cube **never shares this information**.

## 2. Data Use
- Local data is used **exclusively** for authentication and launching the Minecraft game.
- No additional tracking, advertising, or data collection is performed.

## 3. Security
- Mini Cube uses standard practices to secure the local file containing your information.
- However, the developer cannot guarantee absolute security against unauthorized local access.

#4. Affiliation
- Mini Cube **is not affiliated with Mojang or Microsoft**.
- The software does not provide any game files, and the user must legally own Minecraft to use it.
