*A launcher Minecraft with a simple and easy-to-use interface!*

### Project Overview: 
A launcher to be able to launch the Minecraft game on any version, lightweight and open source software.

**I don't plan to release regular updates.**

### Features:
- All versions of Minecraft and automatic updates
- Sign in with a Microsoft account
- Sur le système d'exploitation Windows uniquement (for now)
