## 📖 Overview

**Mini Cube** is a lightweight and open-source Minecraft launcher designed to provide a **simple and easy-to-use interface** for launching the game.

It allows you to launch **any Minecraft version** while keeping the launcher minimal and fast.

⚠️ This project is maintained mainly as a personal project, so **frequent updates are not guaranteed**.

---

## ✨ Features

* Launch **any Minecraft version**
* **Automatic version download**
* **Microsoft account authentication**
* **Discord Rich Presence**
* Lightweight and simple interface
* Open-source

---
